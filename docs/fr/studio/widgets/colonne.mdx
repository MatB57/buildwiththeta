# Colonne

Le widget Colonne est utilisé pour créer une mise en page verticale. C'est l'un des widgets les plus utilisés.

![](https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/Frame%20378.png)

Pour une colonne, le *Main Axis* (axe principal) et le *Cross Axis* (axe transversal) sont deux propriétés de colonne importantes.

L'axe principal s'exécute verticalement et l'axe transversal s'exécute horizontalement. Par défaut, tous les enfants à l'intérieur de la colonne seront disposés verticalement.

## Ajuster l'alignement de l'axe principal

L'axe principal d'un widget de colonne est l'axe vertical, donc l'ajuster changera la façon dont les widgets enfants sont distribués verticalement dans la colonne.

Vous pouvez régler l'alignement de l'axe principal à l'aide des options suivantes :

**Start (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/Start.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants aussi près que possible du début.

**Center (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/center.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants le plus près possible du milieu.

**End (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/end.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants le plus près possible de la fin.

**Space Evenly (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/space%20evenly.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Espacez uniformément les éléments enfants.

**Space Between (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/space%20between.png"
  style="display: inline; max-width: 32px">)**: Placez l'espace libre uniformément entre les enfants.

****Ajustez l'alignement de l'axe transversal****

L'axe transversal d'un widget de colonne est l'axe horizontal, donc l'ajuster changera la façon dont les widgets enfants sont répartis horizontalement dans la colonne.

Vous pouvez régler l'alignement de l'axe transversal à l'aide des options suivantes :

**Start (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/Start.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants aussi près que possible du début.

**Center (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/center.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants aussi près que possible du milieu.

**End (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/space%20evenly.png"
  style="display: inline-block; margin: 0 auto; max-width: 32px">)**: Placez les éléments enfants aussi près que possible de la fin.

**Stretch (<img
  src="https://fftefqqvfkkewuokofds.supabase.co/storage/v1/object/public/theta-assets/docs/column/strech.png"
  style="display: inline; max-width: 32px">)**: Demandez à l'enfant de remplir l'axe transversal (horizontalement).
