# Build with Theta
[Theta](https://buildwiththeta.com) è un tool open source che semplifica la creazione e il test delle app con UI dinamiche, senza la necessità di ribuildarle.

## Per iniziare
Imparate a creare un progetto UI in Theta e a trasmettere un componente all'interno del vostro codice.

### Cosa imparerai
- Come creare un nuovo progetto in Theta.
- Come installare il pacchetto Theta nella vostra applicazione.
- Come eseguire lo streaming di un componente dell'interfaccia utente all'interno del codice.
