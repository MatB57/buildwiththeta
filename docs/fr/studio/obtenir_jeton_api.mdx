# Obtenir le jeton API du projet

<YouTube id="vIE4gZZ8ufo"/>

Pour récupérer le jeton API de votre projet, suivez les étapes ci-dessous :

1. **Créer un nouveau projet :** Accédez à [app.buildwiththeta.com](https://app.buildwiththeta.com) et créez un nouveau projet.

2. **Ouvrez l'éditeur de projet :** Une fois votre projet créé, accédez à l'éditeur de projet.

3. **Cliquez sur le nom du projet :** Vous trouverez le nom du projet dans le coin supérieur gauche, à côté du logo Theta. Clique dessus.

4. **Afficher la clé API :** En cliquant sur le nom du projet, une feuille de fond apparaîtra affichant un champ "Clé API". Ceci est votre jeton API.

Cette clé anonyme (anon) est ce que vous utiliserez pour connecter votre application cliente à Theta, en utilisant notre bibliothèque cliente Flutter. Assurez-vous de stocker cette clé anonyme en toute sécurité, de préférence dans un fichier d'environnement (`.env`), pour empêcher tout accès non autorisé.

Notez que les applications clientes ne peuvent se connecter à votre projet que s'il est défini sur "public". Assurez-vous d'avoir configuré la visibilité de votre projet en conséquence.
