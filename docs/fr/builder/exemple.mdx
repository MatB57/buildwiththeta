# Comment exécuter l'exemple d'application Theta de Flutter

Pour démarrer l'exemple d'application de Theta dans Flutter, accédez simplement au dossier d'exemple dans le package du générateur.
Vous pouvez ensuite exécuter la commande suivante dans votre terminal :

```bash
flutter run -d chrome
```

## Monter le projet

1. À la ligne 9, remplacez l'espace réservé 'thetaKey' par votre propre clé anonyme.
2. À la ligne 35, remplacez le nom de l'UIBox par le nom du composant que vous souhaitez afficher. Par défaut, tous les premiers composants de chaque projet sont nommés page d'accueil.

---

En l'exécutant, vous verrez une page vierge avec le composant créé à l'intérieur de Theta. Avec cette démo, vous pouvez voir comment il se comporte dans une application Flutter, en essayant des modes réactifs et en essayant de mélanger votre propre conception avec du code personnalisé via des workflows (vous trouverez les exemples déjà écrits dans main.dart)

<Warning>Attention : dans ce projet de test, le cache a été désactivé à la ligne 8. Il est toutefois conseillé de le garder toujours actif dans vos projets pour éviter de reconstruire trop souvent l'application.</Warning>

Amusez-vous!
