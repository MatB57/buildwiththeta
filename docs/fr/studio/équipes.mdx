# Équipes
**Theta utilise des équipes pour fournir des expériences personnelles et collaboratives.<br/>Chaque équipe peut être composée d'un seul membre ou de plusieurs utilisateurs.**

Une équipe personnelle peut être utilisée pour des projets de développement individuels et une équipe multi-utilisateurs pour la collaboration avec d'autres membres.<br/>
Chaque projet appartient à une équipe et chaque projet peut avoir différentes branches pour différents environnements ou versions.

Pour partager l'équipe personnelle avec d'autres membres, invitez-les simplement par e-mail à partir du menu "Paramètres de l'équipe" (team settings).
Tous les projets seront automatiquement partagés avec les autres membres invités.
