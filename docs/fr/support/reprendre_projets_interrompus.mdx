# Reprendre les projets en pause

Projet suspendu pour cause d'inactivité ? Pas besoin de s'inquiéter, il ne faut que 3 secondes pour reprendre.

## Didacticiel

<YouTube id="ExmMiu-rAVw" />

1. Accédez à votre projet en pause à partir du [tableau de bord Theta] (https://app.buildwiththeta.com).
2. Dans le coin supérieur gauche, cliquez sur le nom du projet.
3. Activez le champ 'Public'.
4. Sauvegardez.

Votre projet est maintenant accessible via l'API.


## Pourquoi mes projets sont-ils suspendus ?

Parce que nous sommes un petit projet et que nous voulons économiser les ressources système inutilisées pour en tirer le meilleur parti.
