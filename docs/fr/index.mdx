# Theta

## La manière open source de concevoir une interface utilisateur distante

> Créez votre système de conception à distance sans effort, sans écrire de code. Intégrez-le de manière transparente dans votre base de code aux côtés de vos packages préférés, permettant des mises à jour faciles de l'interface utilisateur sans avoir à reconstruire l'intégralité de l'application.

![Theta_extended_negative-large 12](https://github.com/buildwiththeta/buildwiththeta/assets/49411143/ec0a50fc-e946-4750-95c5-1af84776f461)

Theta, un outil hébergé open source, facilite le processus de développement d'applications en permettant des modifications dynamiques de l'interface utilisateur sans nécessiter de reconstruction d'application ou de configurations complexes. Cet outil étendra prochainement le soutien à l'auto-hébergement et au développement local, augmentant encore son accessibilité et sa polyvalence.

Le développement d'applications traditionnelles nécessite souvent des cycles de reconstruction et de déploiement fréquents à chaque changement de code. Bien que certaines solutions existent et soient vraiment bonnes, elles ont souvent une portée limitée. Par exemple, le code push expérimental de Flutter (comme [Shorebird](https://shorebird.dev) pour Android) ne cible que les développeurs experts, tandis que les constructeurs sans code comme [FlutterFlow](https://flutterflow.io) peuvent faire des compromis sur qualité du code tout en nécessitant une programmation supplémentaire pour les fonctionnalités complexes.

## Pourquoi Theta

Conçu pour faciliter la conception rapide de l'interface utilisateur, Theta rationalise la création d'interfaces dynamiques directement dans vos clients d'application. Theta n'est pas un constructeur d'applications sans code. Au lieu de cela, il permet aux développeurs d'intégrer Theta dans leurs flux de travail existants, garantissant une intégration transparente sans compromettre la qualité du code. Avec Theta, vous pouvez concevoir une barre d'application réactive et l'afficher dans une application Flutter en quelques minutes seulement.

Voici quelques-unes des caractéristiques déterminantes de Theta :

### Fonctionnalités disponibles
- **Positionnement absolu** : éléments de conception avec positionnement absolu, similaire à Figma.
- **Design réactif** : créez des éléments et des interfaces réactifs.
- **Intégration Theta** : intégrez Theta dans votre application à l'aide de la bibliothèque Theta Flutter.
- **Mélange code et sans code** : utilisez des flux de travail pour combiner sans code et code, par exemple, exécutez une méthode lorsqu'un nœud est touché.
- **Chargement et gestion des erreurs** : gérez efficacement le chargement et la gestion des erreurs dans les widgets distants.
- **Modes clair et sombre** : prise en charge des thèmes d'interface utilisateur sombres et clairs.
- **Node Overrides** : Remplacez n'importe quel nœud de votre application.

### À venir
- **Collaboration** : Collaborez avec votre équipe.
- **Partage** : partagez vos conceptions avec un lien public.
- **États et paramètres** : gérez les états et les paramètres.

## Dois-je utiliser Theta ?

Theta est un outil permettant d'améliorer l'efficacité de votre processus de développement d'applications, en particulier lorsque vous devez concevoir et intégrer rapidement des composants d'interface utilisateur dynamiques. Il est particulièrement avantageux pour les projets qui nécessitent un mélange de solutions sans code et codées sur mesure.

Cependant, si votre projet nécessite des fonctionnalités très spécifiques qui ne relèvent pas des capacités actuelles de Theta, une solution plus personnalisée pourrait être mieux adaptée. Au fur et à mesure que Theta évolue, davantage de fonctionnalités seront disponibles pour répondre à un plus large éventail d'exigences de projet.

N'oubliez pas que Theta est conçu pour compléter votre workflow de développement existant plutôt que de le remplacer entièrement. Il vaut la peine d'explorer Theta si vous souhaitez réduire le temps consacré à la reconstruction et au déploiement de votre application pour chaque modification de l'interface utilisateur, et si vous appréciez la flexibilité de mélanger le code sans code et le code personnalisé.
